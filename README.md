Inventory Management System

Developed using:
- PHP
- MySQL
- HTML/CSS
- Apache

Features:
- User Authentication
- Product Management
- Supplier Management
- Purchases
- Sales
- Stock Monitoring
- Reports
- Staff Activity Tracking
- Multi-Currency Support

Installation:
1. Import ims_db.sql into MySQL.
2. Copy project folder into htdocs/www.
3. Update database settings in includes/config.php.
4. Start Apache and MySQL.
5. Open the project in your browser.
